Question-3: What is event delegation in DOM?

Event Delegation is a technique where a single event listener is added to a parent element, instead of adding event listeners to multiple child elements individually. The event bubbles up (event bubbling) from the target element to the parent, allowing us to handle events efficiently.

Question-4: What event bubbling in the DOM?

Event Bubbling is a process in JavaScript where an event starts from the target element and propagates up to its ancestors (parent elements). In other words, when an event occurs on a child element, it first triggers on the child, then on its parent, then on the grandparent, and so on until it reaches the document object.
